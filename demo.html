那我们就先聊聊大家熟悉的 CSS border-radius

1.首先，我们看到一个可爱的笑脸，它看上去似乎有点儿方，我们帮它放松一下(要在大屏幕上修改参数)
2.设置一个圆角半径，单位可以是 px，em，%，似乎好一点儿了
3.我们都知道，border-radius 可以输入不止一个值
border-radius: 800px 80px 80px;
4.当我们设置相邻两个圆角的 border-radius 的值之和大于边长时，发现圆角并没有按照我们的预期继续变大 5.那常用的差不多就讲完了

> When the sum of any two adjacent border radii exceeds the size of the border box, UAs must proportionally reduce the used values of all border radii until none of them overlap --[CSS Backgrounds & Borders Level 3](https://www.w3.org/TR/css-backgrounds-3/#corner-overlap) 当相邻两圆角半径之和超出了盒子边长，用户代理（浏览器）必须适当减小所有圆角的半径，直到不再发生重叠

到这里常用的特性差不多就讲完了

现在进入正题

如何制作一个椭圆?

1. 具备长短轴，即 width !== height
2. vertical R / horizonal R === width / height
3. vertical R * 2 === width ; horizonal R * 2 === height
border-radius: 50%;
border-radius: 100px / 120px;

如何制作一个半圆/半椭圆?

1. 50% / 100% 100% 0 0;
2. 50% / 0 0 100% 100%;
3. 0 100% 100% 0 /50%;
4. 100% 0 0 100% /50%;

如何制作一个 1/4 圆或者 1/4 椭圆?

1. 100% 0 0 0;

Global State

Set data-state="something" on a slide and "something" will be added as a class to the document element when the slide is open. This lets you apply broader style changes, like switching the page background.

State Events

Additionally custom events can be triggered on a per slide basis by binding to the data-state name. 


Reveal.addEventListener( 'customevent', function() {
	console.log( '"customevent" has fired' );
} );

Take a Moment

Press B or . on your keyboard to pause the presentation. This is helpful when you're on stage and want to take distracting slides off the screen.
